# AI Automation Playbook

Advanced skills for multi-agent systems, autonomous pipelines, AI vision, speech recognition, scheduling, and workflow automation.

## Skills

| Skill | What it does |
|---|---|
| `hermes-cron-scheduling` | Scheduling conventions and time-window philosophy for Hermes cron jobs |
| `hermes-profile-management` | Manage Hermes profiles — inspect, move cron jobs between profiles, reset auth/state |
| `webhook-subscriptions` | Create event-driven webhook subscriptions so external services trigger agent runs |
| `running-strategic-council` | Run a multi-agent strategic council to pressure-test a business goal before acting |
| `crewai-claude-code-writer` | Wire Claude Code CLI as a builder agent inside a CrewAI crew — solves max_iter exhaustion |
| `crewai-openhands-deploy` | Deploy CrewAI + OpenHands as a self-hosted AI dev stack via Docker Compose |
| `ai-pipeline-feasibility-study` | Validate an AI vision pipeline end-to-end without physical hardware |
| `facility-inspection-ai-playbook` | Design AI-optimized facility inspection playbook for patrol robots and edge cameras |
| `whisper` | Speech recognition — transcription, translation to English, multilingual ASR (99 languages) |
| `graphify` | Build a visual knowledge graph from any documents, code, papers, or notes |

## Prerequisites

- **Core playbook installed**
- **Docker** — for CrewAI / OpenHands skills
- **Node.js 18+** — for webhook platform and graphify
- **Python 3.10+** with venv

## Use Cases

- **Multi-agent planning**: Strategic council debates your goals before you execute
- **Autonomous build pipelines**: CrewAI + Claude Code builds entire Google Sheets / codebases unattended
- **Event-driven automation**: GitHub issues, Stripe payments, IoT sensors trigger your agent automatically
- **AI vision**: Feasibility-test a patrol robot inspection system before buying hardware
- **Voice workflows**: Transcribe meetings, podcasts, or multilingual audio to text
- **Knowledge mapping**: Turn any folder of documents into a navigable knowledge graph
