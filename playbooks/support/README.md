# Support Playbook

Rex is the Support agent. Handles client-facing operations, Lark workspace management, document generation, and file handling.

## Skills

| Skill | Purpose |
|---|---|
| `lark-im-pitfalls` | Pitfalls and scope requirements for Lark IM message access |
| `lark-bitable-schema-setup` | Create and manage Lark Bitable apps, tables, and fields |
| `lark-docx-writer` | Create and write structured Lark Docs via API |
| `reading-lark-files` | Download and read files shared via Lark messages |
| `maps` | Geocode, POIs, routes, timezones via OpenStreetMap/OSRM |
| `google-workspace` | Read/write Google Docs, Sheets, Drive (see core/) |

## Setup

Run `SETUP.md` to install Rex's role-specific skills.
